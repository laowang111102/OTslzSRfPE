# 图书管理系统 java630

## 项目概述

本系统为前后端分离的图书管理系统，实现了管理端与用户端的分离，保证了系统的高效运行与管理。

## 技术栈

### 后端
- ASP.NET Core
- SQL Server
- Entity Framework Core
- JWT

### 前端
- Vue2
- Ant Design of Vue (antd)
- Axios
- Vue Router

## 功能模块

### 管理端
- 用户信息管理：管理用户信息，添加用户
- 密码修改：修改个人密码
- 图书分类管理：添加图书分类
- 图书信息管理：添加图书信息
- 借阅管理：管理借阅人借书情况

### 用户端
- 个人信息修改：修改个人信息
- 密码修改：修改密码
- 借书记录查看：查看个人借书记录

## 系统角色
- 管理员：负责图书与用户的管理
- 普通用户：借阅图书，查看个人信息

## 运行环境
- 后端：.NET Core 运行时环境，SQL Server 数据库
- 前端：现代浏览器
